**Files:**
- Modify: `src/app/(protected)/profile/profile-content.tsx` (or wherever the profile page renders)

The agents/register endpoint is fully functional — it just needs a UI trigger. Add a "Register Agent" button on the profile page that opens a simple form or calls the endpoint directly.

**Step 1: Check what profile page exists**

Read `src/app/(protected)/profile/` to find the content component.

**Step 2: Add a Register Agent section**

Add after the existing profile content:
```typescript
import { Bot, Loader2 } from 'lucide-react';

// Inside the component:
const [registering, setRegistering] = useState(false);

async function handleRegisterAgent() {
  setRegistering(true);
  try {
    const res = await fetch('/api/agents/register', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        name: 'My Agent',
        agentType: 'general',
        capabilities: ['inference', 'prediction'],
      }),
    });
    if (!res.ok) throw new Error('Registration failed');
    // Refresh or show success
  } finally {
    setRegistering(false);
  }
}
```

Add UI:
```tsx
<button
  onClick={handleRegisterAgent}
  disabled={registering}
  className="flex items-center justify-center gap-2 min-h-[44px] rounded-lg bg-chain-hedera text-white text-sm font-semibold w-full active:scale-95 transition-transform"
>
  {registering ? <Loader2 className="w-4 h-4 animate-spin" /> : <Bot className="w-4 h-4" />}
  Register Agent
</button>
```

**Step 3: Verify build**

**Step 1: Get the signing key**

The RP_SIGNING_KEY comes from the World Developer Portal:
1. Go to https://developer.worldcoin.org
2. Select app `app_74d7b06d88b9e220ad1cc06e387c55f3`
3. Navigate to "Sign In with World ID" → "Relying Party"
4. Copy the signing key (private key in hex format)

**Step 2: Add to .env.local**

```bash

```

**Note:** The existing `WORLD_ID_PRIVATE_KEY` in SESSION_CONTEXT.md (`0x96cad...`) may be the same key. Check if the RP signing key in the portal matches this value. If so:
